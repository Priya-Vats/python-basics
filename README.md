## How to Run the Code

1. Clone the repository:

   ```bash
   git clone https://github.com/Priya-Vats/python-basics.git
   ```

2. Navigate into the folder:

   ```bash
   cd python-basics
   ```

3. Run any Python file:

   ```bash
   python filename.py
   ```

> Python 3.x is required.
